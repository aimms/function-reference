GMP::Event::Set
===============

The procedure :aimms:func:`GMP::Event::Set` activates an event.

.. code-block:: aimms

    GMP::Event::Set(
         Event             ! (input) an event
         )

Arguments
---------

    *Event*
        An element in the set :aimms:set:`AllGMPEvents`.

Return Value
------------

    The procedure returns 1 on success, or 0 otherwise.
