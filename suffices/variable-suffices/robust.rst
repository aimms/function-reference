.. _.Robust:

.Robust
=======

Definition
----------

    After solving a robust optimization model, the
    ``.Robust`` suffix contains the solution data of the variable at hand.

Datatype
--------

    The value of the ``.Robust`` suffix is numeric.

Dimension
---------

    The ``.Robust`` suffix has the same dimension and domain as that of the
    variable at hand.

.. note::

    -  See also :doc:`optimization-modeling-components/robust-optimization/index` of the `Language Reference <https://documentation.aimms.com/language-reference/index.html>`__.
